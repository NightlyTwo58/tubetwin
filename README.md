# SYSEN 5530 YouTube digital twin project
Cornell SYSEN 5530 YouTube digital twin project, aiming to build a small-scale testbed that can be used for agentic LLM simulations.  
Build code is in ``src/main`` and data is in ``data/input`` and ``data/output``, respectively.
